# June Dashboard — Pixel-Perfect Clone

Exact 1:1 clone of the June.so marketing screenshot using React.

## 🔄 Apne Repo ko Update Karne ke Steps

### Step 1 — Repo clone karo (agar pehle se nahi hai)
```bash
git clone https://github.com/Vanshiv46/june-dashboard.git
cd june-dashboard
```

### Step 2 — Sirf 2 files replace karo
```
src/App.js   ← naya wala paste karo
src/App.css  ← naya wala paste karo
```

### Step 3 — Run karke dekho
```bash
npm install    # pehli baar
npm start      # localhost:3000 pe khulega
```

### Step 4 — Push karo
```bash
git add src/App.js src/App.css
git commit -m "feat: pixel-perfect june marketing clone"
git push origin main
```

### Step 5 — Vercel auto-deploy
Vercel pe already linked hai → push ke baad automatically deploy ho jayega!
`https://june-dashboard-smoky.vercel.app/`
